Hexo-1-博客搭建基础教程

作为一个程序员，拥有一个属于自己的网站是自己由来已久的梦想（主要是写过的代码老是忘，来回搜很麻烦，所以想有个自己的网站做个记录）。但是限于个人时间和搭建网站所需要的技术栈较多的问题，就采用了框架搭建网站的方式实现，并在此记录下搭建过程。

准备工作

  • windows
  • git
  • node
  • vscode
  • github 账号
  • 域名

git 安装

  • git 官网
  • 默认安装路径在 C:\Program Files\Git
  • 配置用户名和邮箱
  • 配置 ssh key
    • 打开 cmd，输入 ssh-keygen -t rsa -C "你的邮箱"
    • 一直回车，在类似路径中找到 .ssh 文件夹 C:\Users\Administrator\.ssh，其中 Administrator 为你的用户名
    • 记事本打开 id_rsa.pub，复制内容
    • 打开 github，setting -> SSH and GPG keys -> New SSH key，将复制的内容粘贴到 Key 中
    • 打开 cmd，输入以下命令测试连接

node 安装

  • 打开node 官网，也可以在历史版本中选择低一些的版本，避免出现不兼容等问题
  • 下载后默认安装在 C:\Program Files\nodejs，建议不要修改路径，避免出现问题
  • 安装完成后在 cmd 中输入 node -v，看到版本信息后安装成功

  • 进入项目文件夹中安装相关依赖，执行以下命令
